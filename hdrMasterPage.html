Description: This is a module for Sycles (Cycles render integration into Softimage) for quick tweaking of the hdr-environment. This module is similar to the Blender addon Pro-Lighting: Skies

Download: HDRMaster.xsiaddon

How to use: Install as usual addon for Softimage. Restart Softimage after addon installation. Read short introduction document and watch the demo video below for learning basic concepts. To open the property page of the HDR Master object use the same command as to create it (Get - Primitive - Light - HDR Master)
