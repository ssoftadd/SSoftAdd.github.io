Mitsuba and Softimage syncronizator

Last update: 15.10.2017

Back to the main page

Description: This is the set of Python scripts for export scene data (polygon meshes, lights, cameras, shaders etc.) into xml-format for rendering in Mitsuba render.

Download: version 0.8.1

How to use: Install as usual addon for Softimage. For rendering you need to install Visual C++ Redistributable 2013 x64. The addon contains the distributive of Mitsuba, so, you don't need to download it manually. Basic concepts are illustrated on the picture.

Addon contains a set of ICE-compounds for generating point-grids and rendering it as vol-objects. References for these compounds here
