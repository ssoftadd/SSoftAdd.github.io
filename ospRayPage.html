Description: this is a very basic integration of the ospRay render into Softimage. This is quite simple render engine with limited shading system. The main goal of the render is scientific visualizations, but it also can be used for general CG tasks.

Download: addon version 1.0, sources of the addon on the github page

How to use: Install as usual addon for Softimage. Some descriptions are here. Official manual of the render on the ospRay github page
